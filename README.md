This project is a front-end clone of the Halcyonic website template, developed to practice and demonstrate core web development skills using HTML and CSS.

The project focuses on recreating the original design with attention to layout, typography, and responsiveness.

🚀 Features
Fully responsive design
Clean and structured HTML5 code
Styled using modern CSS techniques
Navigation bar and multi-section layout
Image integration and content sections
🛠️ Technologies Used
HTML5
CSS3
🎯 Purpose

This project was created as part of my learning journey in front-end development to improve my skills in layout design, responsiveness, and styling
